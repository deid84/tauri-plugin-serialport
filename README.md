A Tauri v2 plugin for serial port communication, built on top of the [`serialport`](https://crates.io/crates/serialport) crate.

> **Tauri v1 users:** the v1-compatible version is available on the [`v1` branch](https://github.com/deid84/tauri-plugin-serialport/tree/v1).

## Features

- Open / close serial ports with full configuration (baud rate, data bits, parity, stop bits, flow control, timeout)
- Synchronous reads (text and binary)
- Background reader thread that emits Tauri events — subscribe with `listen()`
- Writes (text and binary)
- Control and status lines: RTS, DTR, CTS, DSR, RI, CD
- Buffer operations: `bytesToRead`, `bytesToWrite`, `clearBuffer`
- Break signal: `setBreak` / `clearBreak`
- Disconnection event when the port disappears unexpectedly
- Desktop only (Windows, macOS, Linux)

## Installation

### Rust

Add the plugin to `src-tauri/Cargo.toml`:

```toml
[dependencies]
tauri-plugin-serialport = "2"
```

Register it in `src-tauri/src/main.rs` (or `lib.rs`):

```rust
fn main() {
    tauri::Builder::default()
        .plugin(tauri_plugin_serialport::init())
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

### JavaScript / TypeScript

```bash
npm install tauri-plugin-serialport-api
# or
yarn add tauri-plugin-serialport-api
# or
pnpm add tauri-plugin-serialport-api
```

### Permissions

Add the plugin permission set to your app's capability file (e.g. `src-tauri/capabilities/default.json`):

```json
{
  "permissions": [
    "serialport:default"
  ]
}
```

`serialport:default` enables all 28 commands. Individual `allow-*` / `deny-*` permissions are also available for fine-grained control (see [`permissions/autogenerated/commands/`](permissions/autogenerated/commands/)).

## Usage

```typescript
import { Serialport, availablePorts } from 'tauri-plugin-serialport-api'

// List available ports
const ports = await availablePorts()
console.log(ports) // Record<string, PortInfo>

// Create a port instance
const port = new Serialport({ path: '/dev/ttyUSB0', baudRate: 115200 })

// Open the port
await port.open()

// Subscribe to incoming data (background reader)
await port.listen((data) => {
  console.log('received:', data) // string (decoded with this.encoding)
})
await port.listen((data) => {
  console.log('raw bytes:', data) // Uint8Array
}, false)

// Subscribe to disconnection events
await port.onDisconnected((message) => {
  console.warn('port disconnected:', message)
})

// Start the background reader thread
await port.startListening()

// Synchronous read (blocks up to timeout ms)
const text = await port.read()
const bytes = await port.readBinary()

// Write
await port.write('hello\r\n')
await port.writeBinary(new Uint8Array([0x01, 0x02, 0x03]))

// Change settings on the fly
await port.setBaudRate(9600)
await port.setTimeout(500)

// Signal lines
await port.writeRequestToSend(true)
await port.writeDataTerminalReady(true)
const cts = await port.readClearToSend()

// Buffer info
const pending = await port.bytesToRead()
await port.clearBuffer('Input') // 'Input' | 'Output' | 'All'

// Stop reader and close
await port.stopListening()
await port.close()
```

## API Reference

### Standalone functions

| Function | Description |
|---|---|
| `availablePorts()` | Returns all available ports with metadata (`PortInfo`) |
| `closeAll()` | Closes every open serial port |

### `Serialport` class

#### Constructor options (`SerialportOptions`)

| Option | Type | Default | Description |
|---|---|---|---|
| `path` | `string` | — | Port path (e.g. `COM3`, `/dev/ttyUSB0`) |
| `baudRate` | `number` | — | Baud rate |
| `encoding` | `string` | `'utf-8'` | Text encoding used by `listen()` and `read()` |
| `dataBits` | `DataBits` | `'Eight'` | `'Five'` \| `'Six'` \| `'Seven'` \| `'Eight'` |
| `flowControl` | `FlowControl` | `'None'` | `'None'` \| `'Software'` \| `'Hardware'` |
| `parity` | `Parity` | `'None'` | `'None'` \| `'Odd'` \| `'Even'` |
| `stopBits` | `StopBits` | `'One'` | `'One'` \| `'Two'` |
| `timeout` | `number` | `200` | Read timeout in milliseconds |
| `size` | `number` | `1024` | Read buffer size in bytes |

#### Methods

| Method | Description |
|---|---|
| `open()` | Opens the port |
| `close()` | Closes the port and removes event listeners |
| `listen(fn, decode?)` | Subscribes to background-reader data events |
| `unlisten()` | Removes the data event subscription |
| `onDisconnected(fn)` | Subscribes to port disconnection events |
| `startListening(timeout?, size?)` | Starts the background reader thread |
| `stopListening()` | Stops the background reader thread |
| `cancelRead()` | Sends a stop signal without waiting for thread join |
| `read(timeout?, size?)` | Synchronous text read |
| `readBinary(timeout?, size?)` | Synchronous binary read → `Uint8Array` |
| `write(value)` | Writes a UTF-8 string |
| `writeBinary(value)` | Writes raw bytes |
| `setBaudRate(baudRate)` | Updates baud rate |
| `setDataBits(dataBits)` | Updates data bits |
| `setFlowControl(flowControl)` | Updates flow control |
| `setParity(parity)` | Updates parity |
| `setStopBits(stopBits)` | Updates stop bits |
| `setTimeout(timeout)` | Updates read timeout |
| `writeRequestToSend(level)` | Sets RTS line |
| `writeDataTerminalReady(level)` | Sets DTR line |
| `readClearToSend()` | Reads CTS line |
| `readDataSetReady()` | Reads DSR line |
| `readRingIndicator()` | Reads RI line |
| `readCarrierDetect()` | Reads CD line |
| `bytesToRead()` | Number of bytes waiting in the input buffer |
| `bytesToWrite()` | Number of bytes waiting in the output buffer |
| `clearBuffer(bufferType)` | Clears input, output, or both buffers |
| `setBreak()` | Asserts break signal |
| `clearBreak()` | Clears break signal |

## Events

The background reader thread emits Tauri events. Port path characters `.` and `/` are replaced with `-` in event names.

| Event | Payload | Description |
|---|---|---|
| `plugin-serialplugin-read-{path}` | `{ size: number, data: number[] }` | Data received from the port |
| `plugin-serialplugin-disconnected-{path}` | `string` | Error message when the port disconnects unexpectedly |
